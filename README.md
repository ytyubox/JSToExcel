# JSToExcel

only accept file current

## Installation
```bash
git clone https://github.com/ytyubox/JSToExcel
make -C JSToExcel
```

## Resoure limitation
```json
file: language.json

{
	"en": {
		"hello": "hello",
	},
	"zh": {
	"hello": "你好",
	}
}
```



## Command

```bash
./JSToExcel language.json
```
